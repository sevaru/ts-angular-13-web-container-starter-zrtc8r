ng-morph is a large set of tools for both global code base updates in your project and speeding up your work on Angular schematics. It has ts-morph under the hood and allows you to manipulate with safe TypeScript AST.

In this example you have an Angular project and a ng-morph script in ng-morph-scripts/script.ts folder. Once you explore it and current files, you can run it by adding new Stackblitz terminal and running:

ts-node ng-morph-scripts/script.ts
